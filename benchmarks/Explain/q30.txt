Consider the following OCaml program.

let sales_tax = 1.0
let price_list = [10.0 ; 20.0 ; 5.0]
let add_tax x = sales_tax +. x
let update_price_list price_list = List.map add_tax price_list
let update_sales_tax () = sales_tax *. 2.0

After loading the above expressions, we type into the top-level loop
of OCaml:

# update_sales_tax () ;;

# update_price_list price_list;;

What value will the last expression return?
