The program `shift_point 2` will return `(5, 4)`.

### Explanation:
1. **Initial Point**: The initial point is defined as `(4, 3)`.
2. **Shift Function**: `shift_point n` computes the new point by adding `n` to both the x and y components of the point.
3. **First `shift_point 1`**: 
   - `component_x point + 1` = `4 + 1` = `5`
   - `component_y point + 1` = `3 + 1` = `4`
   - So, `shift_point 1` returns `(5, 4)`.
4. **Renaming `point`**: The variable `point` is reassigned to the result of `shift_point 1`, so now `point` is `(5, 4)`.
5. **Second `shift_point 2`**:
   - `component_x point + 2` = `5 + 2` = `7`
   - `component_y point + 2` = `4 + 2` = `6`
   - So, `shift_point 2` returns `(7, 6)`.
