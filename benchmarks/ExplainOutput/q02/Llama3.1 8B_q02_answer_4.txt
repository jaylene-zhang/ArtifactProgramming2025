**Solution**
```ocaml
let rec repeat (f : 'a -> 'a) : 'a = f (repeat f)
```
**Explanation**
The type of `repeat` is a recursive function that calls itself, so the type variable must be invariant. Since `f` takes an argument of type `'a` and returns a value of type `'a`, the type of `repeat` must be `'a`.
