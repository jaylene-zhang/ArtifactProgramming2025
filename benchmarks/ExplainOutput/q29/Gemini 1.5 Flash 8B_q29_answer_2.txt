**Solution:** `let sum_from_zero f = swap (curry (sumSeries f 0)) (fun x -> x + 1)`

**Explanation:** This correctly uses `curry` to create a function that takes the upper bound as input.  `sumSeries f 0` forms a function that calculates the sum for incrementing steps from 0 to some `upperbound`, resulting in `swap (curry (sumSeries f 0)) (fun x -> x + 1)`.
