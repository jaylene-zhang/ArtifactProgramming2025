**Q4.1**

*   **Base Case:**

    *   `h = Empty`
    *   `add_up Empty + acc = 0 + acc = acc`
    *   `add_up_tr Empty acc = acc`
    *   `Therefore, add_up Empty + acc = add_up_tr Empty acc.`

*   **Step Case:**

    *   `h = Hand (c, h')`

*   **Induction Hypothesis:**

    *   `For all accumulators acc, add_up h' + acc = add_up_tr h' acc.`

*   **Proof:**

    *   `add_up (Hand (c, h')) + acc = (add_up h' + 1) + acc = add_up h' + (1 + acc)`  (by definition of `add_up`)
    *   `By the induction hypothesis, add_up h' + (1 + acc) = add_up_tr h' (1 + acc)`
    *   `add_up_tr h' (1 + acc) = add_up_tr (Hand (c, h')) acc` (by definition of `add_up_tr`)
    *   `Therefore, add_up (Hand (c, h')) + acc = add_up_tr (Hand (c, h')) acc.`

**Q4.2**

*   **Proof:**

    *   `Let acc = 0.`
    *   `By Theorem 1, add_up h + 0 = add_up_tr h 0.`
    *   `Since add_up h + 0 = add_up h, we have add_up h = add_up_tr h 0.`

**Explanation of Approach**

*   **Inductive Proof Structure:**  The proof follows the standard structural induction pattern:
    *   **Base Case:**  Prove the theorem for the simplest possible structure (in this case, an `Empty` hand).
    *   **Step Case:**  Assume the theorem holds for a smaller structure (the induction hypothesis) and then prove it holds for a structure built using that smaller structure (in this case, `Hand (c, h')`).
*   **Theorem Generalization:**  The original statement `add_up h = add_up_tr h 0` is cleverly generalized to `add_up h + acc = add_up_tr h acc`.  This is often necessary for inductive proofs. Introducing the `acc` makes the inductive step easier to prove.
*   **Using Definitions:**  Each step in the proof relies on:
    *   The definitions of the `add_up` and `add_up_tr` functions.
    *   Basic arithmetic properties.
    *   Most importantly, the *induction hypothesis*.
*   **Main Result Proof:** Once the generalized theorem is proven, the very last step simply applies that theorem with `acc = 0` to directly prove the original statement.
